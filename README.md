A scalable backend platform built with raw Node.js,
designed to handle content creation, user management,
and high traffic through scalable architecture.

## Goals

- Build a scalable backend without frameworks
- Understand Node.js internals deeply
- Apply real-world system design concepts

## Tech Stack

- Node.js (raw)
- Custom Router & Middleware
- GitHub for project management
